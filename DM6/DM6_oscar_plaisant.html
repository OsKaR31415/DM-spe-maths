1.

$(E_1) : y' = \dfrac{y}{4}$

a.

(E1) est une équation du type y′ = ay, avec a un réel, donc les solutions de (E1) sont de la forme f(x) = Ceax, avec C une constante réelle, soit, dans le cas de (E1) :


$$f(x) = Ce^{\frac{x}{4}}$$
, avec C une constante réelle.

On cherche donc une valeur de C pour laquelle f(0) = 0, 5, soit :


$$ \begin{array}{rcl} f(0)= 0,5 &\iff& Ce^{\frac{0}{4}} = 0,5\\ &\iff& C = \dfrac{0,5}{e^0}\\ &\iff& C = 0,4 \end{array} $$

on a donc : $f(x) = \dfrac{1}{2}e^{\frac{x}{4}}$

b.

La fonction $x\mapsto \dfrac{x}{4}$ est strictement croissante sur ℝ, et la fonction exponentielle est également strictement croissante sur ℝ. La fonction $x\mapsto e^{\frac{x}{4}}$ est donc aussi strictement croissante sur ℝ.

On sait que $\lim_{t\mapsto +\infty}\left( \dfrac{t}{4} \right) = +\infty$. On sait aussi que limt ↦  + ∞(et) =  + ∞

Donc, on a : $\lim_{t\mapsto +\infty}\left( e^{\frac{t}{4}} \right) = +\infty$

On en déduit que $\lim_{t\mapsto +\infty}\left( \dfrac{1}{2} e^{\frac{t}{4}} \right) = +\infty$, soit que :


limt ↦  + ∞(f(t)) =  + ∞

c.

On cherche la plus petite valeur de t telle que f(t) > 3.


$$ \begin{array}{rcl} f(t) > 3 &\iff& \dfrac{1}{2}e^{\frac{t}{4}} > 3\\ &\iff& e^{\frac{t}{4}} > 6, \text{ et, puisque } e^{\frac{t}{4}} \text{ et } 6 > 0 :\\ &\iff& \frac{t}{4} > \ln{6}\\ &\iff& t > 4\ln{6}\\ \end{array} $$

la population dépasse donc les 300 individus après 4ln 6 jours, soit environ 7 ans et 61 jours.
